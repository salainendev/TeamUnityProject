# TeamUnityProject

## Описание

Данный проект представляет собой игру с видом от третьего лица где нужно собирать бабочек
в данном репозитории представлены основные элементы проекта
для полноценного запуска игры требуется 
дополнительно импортировать набор ассетов : https://assetstore.unity.com/packages/3d/environments/sci-fi/real-stars-skybox-lite-116333
